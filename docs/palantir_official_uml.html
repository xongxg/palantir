1
Connector
«interface» SourceConnector
+ connect(): Connection
+ read(cursor?): Stream<RawRecord>
+ describe(): SourceSchema
+ checkpoint(): Cursor
──▷
Magritte Connector Runtime
«connector» JdbcConnector
-jdbc_url: String
-incremental: Bool
«connector» S3Connector
-bucket: String
-format: Parquet|CSV
«connector» KafkaConnector
-topic: String
-offset: Cursor
──▶
«dataset» RawDataset
+rid: ResourceId
+version: Long
+schema: Schema?
+lineage_id: UUID
CDC 增量同步
Cursor 记录上次同步位置。
下次只拉取 cursor 之后的变更，
避免全量重跑。
2
Transform
«interface» Transform
+ compute(in: Dataset[]): Dataset
+ schema(): Schema
──▷
«service» PipelineGraph
+nodes: Transform[]
+edges: DataFlow[]
+ execute(): Dataset
+ preview(n): Rows
──▶
Built-in Transforms
Filter
predicate: Expr
Join
on: Key[]
Aggregate
groupBy: Col[]
CodeTransform
lang: Python|Spark
──▶
«dataset» CleanDataset
+rid: ResourceId
+upstream_rid: RID
+validated: Bool
Lineage DAG
每个 Dataset 知道自己的
上游 RID。变更上游会
自动 cascade 触发重算。
3
Ontology
Ontology Schema (Design-time)
«objectType» ObjectType
+apiName: String
+primaryKey: Property
+properties: Property[]
+datasource: DatasetRID
+ validate(obj): Bool
── 1..*
«linkType» LinkType
+from: ObjectType
+to: ObjectType
+cardinality: ONE_MANY|MANY
+foreignKey: Property
──▶
Object Storage (Runtime)
«service» ObjectStorage
-backend: Cassandra|Foundry
+ upsert(obj): void
+ get(id): Object
+ delete(id): void
+ link(from,to,type): void
──▶
Object Graph
«instance» OntologyObject
+objectRid: RID
+type: ObjectType
+properties: Map<K,V>
+links: Link[]
+version: Long
Bi-temporal
每个对象有 valid_time
（业务时间）+ tx_time
（写入时间），支持
历史状态查询。
4
Semantic Query
«service» ObjectSetAPI
+objectType: ObjectType
+ filter(pred): ObjectSet
+ navigate(linkType): ObjectSet
+ groupBy(prop): GroupedSet
+ aggregate(fn): Result
+ load(props[]): Object[]
──▶
«value» ObjectSet
// lazy evaluated
// AST, not rows
+ filter(): ObjectSet
+ intersect(): ObjectSet
+ union(): ObjectSet
+ execute(): Result
──▶
«service» QueryBackend
-engine: Spark|Arrow
+ translate(ast): SQL/Spark
+ execute(): DataFrame
+ cache(): void
例：语义查询
Employee.filter(
  spend_ratio > 30%
).navigate(
  HAS_MANAGER
).load([name])

Functions
«interface» OntologyFunction<T>
+ apply(objectSet): T
+ returnType(): Schema
+ isAggregation(): Bool
──▷
Function Types
«aggregation» AggFunction
// SUM / AVG / COUNT
// MAX / PERCENTILE
+ groupBy(linkType)
«derived» DerivedProperty
// virtual property
// computed on-read
«rule» BusinessRule
// risk_level()
// compliance_flag()
──▶
«result» FunctionResult
+value: Any
+object_rid: RID?
+computed_at: Timestamp
TypeScript / Python
Function 代码部署到
Foundry Runtime，不需要
管理服务器。结果可
缓存为 Derived Property。
